## 什么是 SceneSelf？

SceneSelf 是一个 **AI 场景照片生成器**。上传一张自拍，用一句话描述场景，即可获得 **6 张连贯、竖版（4:5）、手机随手拍风格的场景照片**——旅行 photo dump、节日照片、幻想角色集锦等。这是一款以创意和想象为核心的产品，底层是完整的生产级技术栈：用户认证、订阅计费、积分系统、兑换码、管理后台和国际化。

### 核心特性

- **场景生成** — 由 OpenRouter（Gemini）驱动的 6 张成组写真，内置质量、身份、组一致性三道检查
- **用户认证** — 邮箱密码 + Google OAuth（Better Auth）
- **积分计费** — 按次计费的积分系统 + 可审计的积分账本 + 订阅计划（Creem 支付）
- **兑换码系统** — 管理员批量生成 + 合作伙伴 API + 用户端 `/redeem` 核销
- **管理后台** — 用户管理、积分调整、订阅控制、兑换码批次、API key 管理
- **国际化** — 完整的中英文支持（next-intl）
- **邮件系统** — 交易邮件（Resend）：验证、密码重置、购买确认
- **营销页面** — 首页、定价、博客（MDX）、法律页面
