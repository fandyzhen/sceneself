## 修改定价计划

编辑 `constants/billing.ts`。上线配置为三档（`weekly`/`monthly`/`yearly`），均 `per_cycle` 一次性发放：

```typescript
monthly: {
  priceCents: 2900,                                  // $29
  creditsPerCycle: 8000,
  cycle: "month",
  creemPriceId: process.env.CREEM_MONTHLY_PRICE_ID,  // 对应 Creem 产品 ID
  grantSchedule: { mode: "per_cycle" },
}
```

定价页"最受欢迎"高亮由 `RECOMMENDED_PLAN` 控制。

## 修改场景计费

编辑 `lib/scene/pricing.ts` 中的常量：

| 配置 | 变量 | 默认值 |
|------|------|--------|
| 每组照片数 | `SHOTS_PER_SET` | 6 |
| 每张积分 | `CREDITS_PER_PHOTO` | 50 |

一组消耗 `SHOTS_PER_SET × CREDITS_PER_PHOTO` = 300 积分；未交付的张数自动返还。

## 替换 AI 提供商

场景流程默认基于 OpenRouter（出图 Nano Banana 2 + 文本/视觉 Gemini Flash Lite）。如需更换提供商：

1. 替换或扩展 `lib/openrouter/` 中的客户端封装（或新建 `lib/<provider>/`）
2. 更新 `lib/scene/services/` 中的调用点（`image-gen.ts`、`quality-check.ts`、`identity-check.ts`、`scene-planner.ts`、`story-line.ts`）
3. 更新 `lib/scene/config.ts` 中的 `imageModel` / `textModel` / `visionModel`
4. 相应更新 `OPENROUTER_*` 环境变量
5. 保持相同的积分扣除与返还模式

## 添加新语言

1. 复制 `messages/en.json` 到 `messages/XX.json`（新语言代码）
2. 翻译所有字符串
3. 在 `i18n.config.ts` 中添加语言：

```typescript
export const locales = ['en', 'zh', 'XX'] as const;
```

4. 更新 `proxy.ts` 和 `lib/i18n.ts`

## 自定义主题

颜色在 `app/globals.css` 中使用 CSS 自定义属性定义：

```css
:root {
  --background: 0 0% 100%;
  --foreground: 0 0% 3.9%;
  --primary: 0 0% 9%;
  /* ... */
}
```

深色模式变量在 `.dark` 类中。

## 添加新的受保护页面

1. 在 `app/[locale]/(protected)/your-page/page.tsx` 创建新页面
2. 布局的会话检查会自动保护该页面
3. 在 `features/navigation/config.ts` 中添加导航
