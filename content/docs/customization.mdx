## Modify Pricing Plans

Edit `constants/billing.ts`. The launch config exposes three tiers (`weekly`, `monthly`, `yearly`), all `per_cycle`:

```typescript
monthly: {
  priceCents: 2900,                                  // $29
  creditsPerCycle: 8000,
  cycle: "month",
  creemPriceId: process.env.CREEM_MONTHLY_PRICE_ID,  // Match your Creem product ID
  grantSchedule: { mode: "per_cycle" },
}
```

The recommended plan is highlighted on the pricing page via `RECOMMENDED_PLAN`.

## Modify Scene Pricing

Edit the constants in `lib/scene/pricing.ts`:

| Setting | Variable | Default |
|---------|----------|---------|
| Photos per set | `SHOTS_PER_SET` | 6 |
| Credits per photo | `CREDITS_PER_PHOTO` | 50 |

One set costs `SHOTS_PER_SET × CREDITS_PER_PHOTO` = 300 credits; undelivered photos are refunded automatically.

## Replace AI Provider

The Scene flow is built on OpenRouter (Nano Banana 2 for images + Gemini Flash Lite for text/vision). To switch providers:

1. Replace or extend the client wrappers in `lib/openrouter/` (or add a new `lib/<provider>/`)
2. Update the call sites in `lib/scene/services/` (`image-gen.ts`, `quality-check.ts`, `identity-check.ts`, `scene-planner.ts`, `story-line.ts`)
3. Update `imageModel` / `textModel` / `visionModel` in `lib/scene/config.ts`
4. Update the `OPENROUTER_*` environment variables accordingly
5. Keep the same credit deduction and refund pattern

## Add a New Language

1. Copy `messages/en.json` to `messages/XX.json` (new locale)
2. Translate all strings
3. Add locale to `i18n.config.ts`:

```typescript
export const locales = ['en', 'zh', 'XX'] as const;
```

4. Update `proxy.ts` and `lib/i18n.ts`

## Customize Theme

Colors are defined in `app/globals.css` using CSS custom properties:

```css
:root {
  --background: 0 0% 100%;
  --foreground: 0 0% 3.9%;
  --primary: 0 0% 9%;
  /* ... */
}
```

Dark mode variables are in the `.dark` class.

## Add New Protected Pages

1. Create a new page in `app/[locale]/(protected)/your-page/page.tsx`
2. It's automatically protected by the layout's session check
3. Add navigation in `features/navigation/config.ts`
