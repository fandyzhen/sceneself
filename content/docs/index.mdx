## What is SceneSelf?

SceneSelf is an **AI scene photo set generator**. Upload one selfie, describe a scene in one sentence, and get **6 cohesive, portrait (4:5), phone-shot style photos** — a travel grid, a holiday photo dump, a fantasy character set, and more. It is a creative, imagination-first product built on a full production stack: authentication, subscription billing, credits, redemption codes, admin dashboard, and i18n.

### Key Features

- **Scene Generation** — 6-photo cohesive scene sets powered by OpenRouter (Gemini), with quality, identity, and set-coherence checks
- **Authentication** — Email/password + Google OAuth via Better Auth
- **Credit-Based Billing** — Per-use credit system with auditable ledger + subscription plans via Creem
- **Redemption Codes** — Admin-generated code batches + partner API + user redemption at `/redeem`
- **Admin Dashboard** — User management, credit adjustment, subscription control, code batches, API keys
- **Internationalization** — Full English and Chinese support via next-intl
- **Email System** — Transactional emails via Resend (verification, password reset, purchase confirmation)
- **Marketing Pages** — Landing page, pricing, blog (MDX), legal pages
