# SceneSelf

SceneSelf turns a single selfie into a cohesive, phone-shot style scene album in minutes — no studio, no shoot, no waiting.

## Highlights

- AI scene album generator with curated scene and photo-dump presets
- Credit-based billing with weekly / monthly / yearly subscriptions
- Production-grade Next.js 16 + React 19 + Drizzle ORM stack
- Built-in auth (Better Auth + Google OAuth), payments (Creem), email (Resend)
- Internationalized (zh / en) with App Router locales
- Admin dashboard for user, subscription and credit management

## Quickstart

```bash
git clone https://github.com/sceneself/sceneself.git
cd sceneself
pnpm install
cp .env.example .env  # fill in your credentials
pnpm db:push
pnpm dev
```

See `content/docs/quickstart.mdx` for the full setup walkthrough.
